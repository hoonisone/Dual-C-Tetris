# Dual-C-Tetris
This is a Tetris game implemented in C Language (Dual Mode)

# How To Play
* user1
  * a: move left one space
  * d: move right one space
  * w: spin clockwise
  * s: drop one block
  * Tab: hard drop
    
* user2
  * ←: move left one space
  * →: move right one space
  * ↑: spin clockwise
  * ↓: drop one block
  * Tab: hard drop
 
* Common
  * r: restart only when anyone is over.
